# Installing

You can install REAVIZ with [NPM](https://www.npmjs.com/package/reaviz) or Yarn.

NPM
```
npm install reaviz --save
```

-- or --

Yarn
```
yarn add reaviz
```

You can try it before you commit on [CodeSandbox](https://codesandbox.io/embed/m7rl2z1989).

REAVIZ is compatible with React v16+ and works with ReactDOM. React Native
is not supported at this time.
