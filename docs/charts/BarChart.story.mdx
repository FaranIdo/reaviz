# Bar Chart

---
<br />
<Story id="demos-bar-chart-vertical-single-series--simple" />
<br />

A bar chart is a chart with rectangular bars with lengths proportional to the
values that they represent.  One axis of the chart shows the specific categories being compared,
and the other axis represents a discrete value.

Bar charts provide a visual presentation of categorical data.
Categorical data is a grouping of data into discrete groups, such as months of the year,
age group, shoe sizes, and animals. These categories are usually qualitative.
Bars on the chart may be arranged in any order. Bar charts are useful for comparisons and trend over time.

Types supported by reaviz:

- Single Series Vertical / Horizontal
- Multi Series Vertical / Horizontal
- Stacked Vertical / Horizontal
- Stacked Normalized Vertical / Horizontal
- Stacked Diverging Vertical / Horizontal
- Marimekko
- Radial
- Sparkline
- Waterfall
- Histograms

## Quick Start
To create a Bar chart, use import the `BarChart` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

```jsx
import { BarChart } from 'reaviz';

const MyChart = () => (
  <BarChart
    height={300}
    width={300}
    data={[
      { key: 'DLP', data: 13 },
      { key: 'SIEM', data: 2 },
      { key: 'Endpoint', data: 7 }
    ]}
  />
)
```

In the simple example, the data object is just a `Array` of `key` and `data`. Below
you can learn more about how to pass more complex types.

## API

Below is the list of props the bar chart accepts.

| Prop  | Description | Type | Required | Default |
| --- | --- | --- | --- | --- |
| `height` | The height of the chart | `number` | `false` | `null` |
| `width` | The width of the chart | `number` | `false` | `null` |
| `margins` | Margins around the chart | `number` | `false` | `20` |
| `data` | The data that is used to build the chart | `ChartDataShape[]` | `true` | `[]` |
| `className` | Additional class names to apply to the container | `string` | `false` | `null`  |
| `style` | Additional styles names to apply to the container | `Object` | `false` | `null`  |
| `series` | The bar series configuration | `JSX.Element` | `true` | `[]`  |
| `yAxis` | The y-axis configuration | `JSX.Element` | `true` | `<LinearYAxis />`  |
| `xAxis` | The x-axis configuration | `JSX.Element` | `true` | `<LinearXAxis />`  |
| `gridlines` | The gridline configuration | `JSX.Element` | `false` | `null`  |
| `brush` | The brush configuration | `JSX.Element` | `false` | `null`  |
| `secondaryAxis` | Additional axis configuration | `JSX.Element[]` | `false` | `null`  |

of data that are categorical in nature. This is the most simple chart type but can
