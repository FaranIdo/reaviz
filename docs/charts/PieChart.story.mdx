# Pie Chart

---
<br />
<Story id="demos-pie-chart-pie--simple" />
<br />

A pie chart is divided into sectors, illustrating numerical proportion.
In a pie chart, the arc length of each sector (and consequently its central angle and area),
is proportional to the quantity it represents. While it is named for its resemblance to a pie
which has been sliced, there are variations on the way it can be presented.

## Quick Start
To create a Bar chart, use import the `PieChart` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

```jsx
import { PieChart } from 'reaviz';

const MyChart = () => (
  <PieChart
    height={300}
    width={300}
    data={[
      { key: 'Bacon', data: 13 },
      { key: 'Eggs', data: 13 },
      { key: 'Cheese', data: 13 },
    ]}
  />
);
```

## API

Below is the list of props the chart accepts.

| Prop  | Description | Type | Required | Default |
| --- | --- | --- | --- | --- |
| `height` | The height of the chart | `number` | `false` | `null` |
| `width` | The width of the chart | `number` | `false` | `null` |
| `margins` | Margins around the chart | `number` | `false` | `20` |
| `data` | The data that is used to build the chart | `ChartDataShape[]` | `true` | `[]` |
| `className` | Additional class names to apply to the container | `string` | `false` | `null`  |
| `style` | Additional styles names to apply to the container | `Object` | `false` | `null`  |
| `series` | The area series configuration | `JSX.Element` | `true` | `[]`  |
